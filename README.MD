# 📊 Программа для обработки Excel-файлов

Программа предназначена для автоматизации обработки Excel-файлов и подготовки списков поступающих.

---

## 📁 Структура проекта

Проект использует несколько рабочих каталогов:

### `source/`

Содержит исходные Excel-файлы, которые необходимо обработать.

### `temp/`

Используется для хранения промежуточных результатов обработки (например, данных, полученных с Госуслуг).

### `ready/`

Содержит итоговые обработанные списки поступающих, подготовленные для работы с системой **1С**.

---

# 🚀 Запуск программы

## Требования

Перед запуском программы убедитесь, что на компьютере установлен **Python**.

Также необходимо установить все зависимости, указанные в файле `requirements.txt`:

```bash
pip install -r requirements.txt
```

---

## Запуск

Запустите исполняемый файл:

```bash
python priemka/splitter/main.py
```

Файл `main.py` является основной точкой входа в программу и отвечает за запуск процесса обработки.

---

## 📂 Результат работы

После завершения обработки:

* исходные файлы остаются в папке `source/`;
* промежуточные данные сохраняются в папке `temp/`;
* готовые обработанные файлы появляются в папке `ready/`.
